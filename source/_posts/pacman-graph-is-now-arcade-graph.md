Big update: **Pacman Contribution Graph** has evolved into **Arcade Contribution Graph**.

What started as one Pacman game is now a growing arcade of contribution-graph games for GitHub and GitLab.

<!--more-->

## From One Game to an Arcade

Pacman Contribution Graph began with one goal: make contribution graphs more fun and alive.

Now the project supports multiple arcade experiences:

- **Pac-Man**
- **Breakout**
- **Galaga**
- **Puzzle Bobble**
- **Bomberman**
- **Minesweeper**

And this is only the beginning.

Project repo: [github.com/abozanona/pacman-contribution-graph](https://github.com/abozanona/pacman-contribution-graph)
